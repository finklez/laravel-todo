# laravel-todo

## Instructions:

- Change dir

      cd backend
    
- Install packages

      composer install
    
- Update key (automatically added to .env)

      php artisan key:generate
    
- Update DB credentials in .env

      DB_DATABASE=
      DB_USERNAME=
      DB_PASSWORD=
      
- Run migrations
      
      php artisan migrate
    
- Run server

      php artisan serve
        
- Browse local server

      http://localhost:8000
